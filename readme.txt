Página Web de Adopción de Mascotas

Esta es una página web sencilla diseñada para promover la adopción de mascotas, destacando a los perros disponibles para adopción, proporcionando información de contacto y detalles sobre la organización.

Estructura del Proyecto

index.html: Página principal de la web.

style.css: Hoja de estilos para darle una apariencia atractiva y amigable.

script.js: Archivo JavaScript para funcionalidades interactivas como el carrusel de imágenes y videos.

img/: Carpeta que contiene las imágenes y videos de las mascotas.
Descripción del HTML

Encabezado 
Título: Establece el título de la página web como "Adopción de mascotas".

Hoja de Estilos y Script: Enlaza la hoja de estilos (style.css) y el archivo JavaScript (script.js).
